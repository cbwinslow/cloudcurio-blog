API Endpoints

The system exposes RESTful APIs for all functionality:

Agent Management

GET /api/agents - List all agents
POST /api/agents - Create a new agent
PUT /api/agents/{id} - Update an agent

Research

POST /api/research/start - Start research session
GET /api/research/sessions - List research sessions

RAG System

POST /api/rag/search - Search knowledge base
POST /api/rag/documents - Add document

Blog Management

GET /api/blog - List blog posts
POST /api/blog/generate-from-research/{id} - Generate from research
